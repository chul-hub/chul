# chul

This repository contains a simple plane game using **pygame**. The player controls a green plane, shoots bullets with the space key, and avoids red enemy planes.

## Requirements
- Python 3.11+
- `pygame` library (`pip install pygame`)

## Running the Game
```
python3 plane_game.py
```
Use the arrow keys to move and the space bar to shoot. Close the window or collide with an enemy to end the game.
