# 스페이스 리듬 벽돌깨기

웹 브라우저에서 즐길 수 있는 빠른 템포의 벽돌깨기 게임입니다. 화려한 배경과 다양한 강화 아이템으로 클래식한 벽돌깨기에 리듬 아케이드 감성을 더했습니다.

## 특징

- 3개의 독창적인 레벨 패턴과 점점 높아지는 난이도
- 패들 확장, 슬로모션, 멀티볼 등 다양한 강화 아이템
- 별빛이 흐르는 우주 무대와 파티클 이펙트
- 키보드(← →)와 마우스/터치 모두 지원
- 스페이스바로 일시 정지, R 키로 재시작 가능

## 실행 방법

1. 저장소를 클론한 뒤 `index.html` 파일을 브라우저로 직접 열어주세요.
2. 또는 간단한 정적 서버를 이용하려면 다음 명령을 실행하세요.

   ```bash
   python -m http.server 8000
   ```

3. 브라우저에서 `http://localhost:8000`에 접속해 게임을 즐겨보세요.

## 조작법

- **← → 방향키**: 패들을 좌우로 이동합니다.
- **마우스/터치**: 캔버스 위에서 움직이면 패들이 따라옵니다.
- **스페이스바**: 게임을 일시 정지하거나 재개합니다.
- **R 키**: 게임 오버 또는 올클리어 후 새로운 게임을 시작합니다.

즐거운 게임 플레이를 위해 브라우저의 사운드를 켜고, 자신만의 리듬으로 벽돌을 깨보세요!
